## Secure File Encryption System

The **Secure File Encryption System** is a Java-based application that allows users to securely encrypt and decrypt files using password-based cryptography. The system ensures **data confidentiality and integrity** using strong encryption algorithms.

## Technologies Used

* Java
* Java Swing (GUI)
* Java Cryptography Architecture (JCA)

##  Cryptographic Techniques Used

* AES (Advanced Encryption Standard) for file encryption
* SHA-256 for file integrity verification
* PBKDF2 (Password-Based Key Derivation Function) for secure key generation
* Random Salt for enhanced security

## Features

* Strong encryption using AES
* Password-based key derivation using PBKDF2
* Random salt to prevent brute-force attacks
* SHA-256 hashing for integrity verification


This project demonstrates the implementation of **secure file encryption using cryptographic principles**. It ensures confidentiality, integrity, and usability, making it suitable for protecting sensitive data.
